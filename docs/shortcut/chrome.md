# GoogleChromeのショートカットキー

## 注意事項

MacとWindowsキーが混在する。  
内容が増えたら分けて記載するようにする。

カテゴライズも増えてから考える。

## 一覧

|概要|Mac|Windows|
|-|-|-|
|新規ウィンドウ|`command（⌘）` + `N`||
|新規シークレットウィンドウ|`command（⌘）` + `shift（⇧）` + `N`||
|新規タブ|`command（⌘）` + `T`||
|タブ前後移動|`command（⌘）` + `option（⌥）` +  + `左（◀）` or `右（▶）`||
|アドレスバーにフォーカス|`command（⌘）` + `L`||
