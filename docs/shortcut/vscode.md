# VisualStudio Codeのショートカットキー

## 注意事項

MacとWindowsキーが混在する。  
内容が増えたら分けて記載するようにする。

カテゴライズも増えてから考える。

## 本体の操作

|概要|Mac|Windows|
|-|-|-|
|新規ウィンドウ|`command（⌘）` + `shift（⇧）` + `N`||
|新規ターミナル|`control（⌃）` + `shift（⇧）` + ``@（`）``||
