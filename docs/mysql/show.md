# DATABASEとTABLEの一覧表示

よく`s`を忘れてシンタックスエラーになる。

## DATABASE一覧

```sql
SHOW DATABASES;
```

## TABLE一覧

```sql
SHOW TABLES <DATABASENAME>;
```

:::tip
DBスキーマを指定してるのであればDBスキーマ名は不要
:::
